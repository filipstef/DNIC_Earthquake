About this page

This is a educational site for earthquake safety. This site warns about recent earthquakes and informs users how to act in the event of a earthquake. This site features a few pages:

  • Home - Has a list of recent earthquakes, and upon clicking a element from the list there is a poopup with more information
  • Learn - short videos with quizes intended to teach users how to act in the event of a earthquake
  • About us - Information about this site (current page)
